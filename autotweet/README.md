# Requirements
- node.js

# Set up
1. install twitter module
```
npm install twitter
```

2. copy .text.txt
```
cp .text.txt text.txt
```

3. edit text.txt
 write down what you want tweet.

4. copy .keys.json
```
cp .keys.json keys.json
```

5. edit keys.json  
copy consumer key, consumer secret, access token key and access token secret to keys.json

6. tweet
```
node autotweet.js
```
